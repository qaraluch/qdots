# bloodtrees rice

Installation through script:
install-rice

It reads global variables:
- QDOTS_RICE_DIR
- QDOTS_RICE_THEME_SOURCE_NAME
- QDOTS_RICE_PATH
- QDOTS_RICE_XRESOURCES
- QDOTS_RICE_SCRIPT
- QDOTS_RICE_SEQUENCES

path:
```
~/.local/share/rice/bloodtrees
```
